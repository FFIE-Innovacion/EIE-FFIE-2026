# EIE-FFIE-2026
Dashboard IEIE FFIE 2026 — tablero interactivo del Índice de Estado de Infraestructura Educativa de las sedes oficiales de Colombia. Resumen nacional/territorial, análisis por dimensión, ficha territorial, mapas interactivos y metodología. Sitio estático (HTML/CSS/JS + Leaflet/ECharts) publicado con GitHub Pages.
